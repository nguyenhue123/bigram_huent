# Vietnamese Bigram Language Model

Project này xây dựng **mô hình ngôn ngữ Bigram tiếng Việt (mức âm tiết)** bằng Python.

Chương trình có các chức năng:

- Huấn luyện **Bigram Language Model**
- Tính **xác suất của một câu**
- Tính **log probability**
- **Sinh câu tiếng Việt** từ mô hình

---

## Yêu cầu

Python 3.8+

Không cần thư viện ngoài (chỉ dùng thư viện chuẩn của Python).
